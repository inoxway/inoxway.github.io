Myxsi es un panel de administración completo con terminal REAL, gestor de archivos, integración con Immich y un asistente de voz inteligente. Todo desde tu navegador.

Ver en GitHub Ver Demo

✨ Características Principales

Terminal REAL

Terminal Linux completa Ejecuta bash via web, vim, htop y cualquier comandos Linux debian.

bash/zsh soporte

Autenticación PAM

Usa las mismas credenciales de tu sistema Linux. No almacenamos contraseñas, todo es verificado contra el sistema operativo.

Seguridad máxima

Immich Integration

Google Photos alternativo. Backup automático de fotos, reconocimiento facial, mapas y álbumes compartidos.

Self-hosted

Gestor de Archivos

Navega, sube, descarga y organiza tus archivos desde el navegador. Soporte para múltiples formatos y vista en cuadrícula.

Drag & drop

Asistente de Voz

Controla tu servidor con comandos de voz. "Myxsi, mostrar estado del sistema", "Myxsi, reiniciar servidor" y más.

Próximamente

Monitoreo en Tiempo Real

CPU, RAM, disco, procesos y servicios en tiempo real. Gráficos interactivos y alertas configurables.
